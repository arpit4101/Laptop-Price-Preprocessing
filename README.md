# Laptop-Price-Preprocessing
End-to-end data cleaning &amp; feature engineering pipeline on a raw, uncleaned laptop price dataset (1,303→1,273 rows). Covers missing value diagnosis, IQR outlier analysis, grouped imputation, column decomposition (CPU/Memory/Resolution), hybrid encoding (ordinal/OHE/label), MinMax scaling, baseline regression validation (R² 0.84 with Random Forest)
